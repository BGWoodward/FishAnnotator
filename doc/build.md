General build instructions
==========================

Dependencies
------------

FishAnnotator uses the following 3rd party libraries:

* [Qt5][Qt5]
* [Boost][Boost]
* [OpenCV][OpenCV]
* [FFMPEG][FFMPEG]

[CMake][CMake] is also required to build.
