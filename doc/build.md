General build instructions
==========================

Dependencies
------------

FishAnnotator uses the following 3rd party libraries:

* [Qt5][Qt5] 5.8.0
* [Boost][Boost] 1.62
* [FFMPEG][FFMPEG] 3.3.2

[CMake][CMake] 3.5.1+ is also required to build.
