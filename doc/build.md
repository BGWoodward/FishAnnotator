General build instructions
==========================

Dependencies
------------

Tator uses the following 3rd party libraries:

* [Qt5][Qt5] 5.9.4
* [Boost][Boost] 1.62
* [FFMPEG][FFMPEG] 3.3.2

[CMake][CMake] 3.5.1+ is also required to build.
